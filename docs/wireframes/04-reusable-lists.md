**Contains:** pinned lists, active counts, last used, New List.

```text
┌─────────────────────────────────────┐
│ Lists                         +     │
│ PINNED                              │
│ Groceries             6 active     │
│ Costco                3 active     │
│                                     │
│ OTHER LISTS                         │

├─────────────────────────────────────┤
│ Today  Areas   [+]   Lists   Tasks  │

```

## 30. Create or Edit List

**Contains:** name, icon, reusable and memory settings, sorting.

```text
┌─────────────────────────────────────┐
│ ← New List                    Save  │
│ List name                           │
│ [ Groceries                       ] │
│ Icon               Shopping cart  › │
│ Reusable list                [ ON ] │
│ Remember completed items     [ ON ] │
│ Suggest previous items       [ ON ] │
│ Show last-used dates         [ ON ] │
│ Default sorting            Manual › │
└─────────────────────────────────────┘
```
