# Screens and Rough ASCII Wireframes

This index covers the complete Nudge screen inventory: full screens, sheets, intervention states, widgets, empty states, and system surfaces. The ASCII layouts are structural wireframes, not final visual designs.

## Navigation model

```text
┌─────────────────────────────────────┐
│              SCREEN                 │
├─────────────────────────────────────┤
│ Today  Areas   [+]   Lists   Tasks  │
└─────────────────────────────────────┘
```

Secondary destinations are available from More or the Today overflow menu:

- Search
- Gemini
- History
- Insights
- Templates
- Archived Items
- Settings
- Help and Feedback

## Scope summary

- 38 primary full-screen concepts
- 12 important sheets and modal states
- 5 widget layouts
- 6 empty, error, and feedback states
- 73 numbered screen or state concepts in total

Some designs will share implementation components. Add and Edit use the same form, Gemini Processing is a state of Gemini Input, Shopping Session can be a mode of List Detail, and completion confirmation may be an overlay instead of a permanent route.
